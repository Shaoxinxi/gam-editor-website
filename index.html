FAQ

Answers to common questions

Do I need to purchase the game to use this editor? +

Yes. This editor is only for modifying game assets you have legally obtained. Please ensure you own a valid license for the game.

Will modified files affect multiplayer? +

No impact in single-player mode. In multiplayer, some servers may detect and restrict modified vehicles. We recommend using it in single-player only.

Will my modifications be lost after a game update? +

Major game updates may overwrite modified files. We recommend re-exporting your save after each game update, then using the Load Save feature to restore previous modifications.

Can the export limit be removed? +

Upgrading to a membership unlocks unlimited exports, along with advanced features like saves and snapshots.

Does it support other games? +

Currently supports SnowRunner and BeamNG.Drive at launch. Support for more games will be added gradually.

"Game installation directory not found" after switching games? +

Please add the game in the GAM main application first. The editor reads game paths from GAM configuration. For BeamNG.Drive, make sure the mod deployment path is configured correctly.

The editor cannot detect installed mods? +

For BeamNG.Drive, make sure the mod deployment path is set. Other games set it automatically after being added, so no manual configuration is needed.

No changes appear in-game after editing? +

After editing, save the parameters first, then click "Export to Game". Make sure the game is closed before exporting, otherwise the export will have no effect.

How is GAM Vehicle Editor different from editing game files directly? +

Directly editing game files requires locating config files and writing code - error-prone and hard to undo. GAM Vehicle Editor provides a visual interface, auto-backup before editing, and one-click restore, greatly reducing the risk of mistakes.

How do I use the SnowRunner / BeamNG.Drive vehicle editor? +

After adding the game in GAM mod manager, click the "Editor" button in the top menu to launch it. The editor scans all vehicles automatically. Select a vehicle to adjust parameters, then click "Export to Game" to apply changes instantly.

Which tool is recommended for modifying BeamNG.Drive vehicles? +

GAM Vehicle Editor supports visual editing of BeamNG.Drive vehicle parameters - suspension, engine, tires - with auto-backup before changes, never corrupting original game data, and supporting reset to defaults at any time.
